

# Путь в Агро 🌱

Профориентационный веб-навигатор для школьников, студентов и абитуриентов, предназначенный для выбора профессий агропромышленного комплекса и построения образовательной траектории.

---

## 📌 Описание проекта

**«Путь в Агро»** — это веб-приложение, разработанное на языке Python с использованием фреймворка Django.

Проект помогает пользователям:

- ознакомиться с профессиями агропромышленного комплекса;
- пройти профориентационный тест;
- получить персонализированные рекомендации по профессиям;
- изучить образовательные учреждения;
- просмотреть статьи и информационные материалы об АПК.

Проект ориентирован на:
- школьников 8–11 классов;
- студентов;
- абитуриентов;
- всех, кто интересуется аграрной сферой и современными агротехнологиями.

---

## ⚙️ Основные функции системы

### 🏠 Главная страница
- описание проекта и его цели;
- визуальный маршрут пользователя;
- информационные блоки и статьи об АПК.

---

### 🧠 Профориентационный тест
- последовательное прохождение вопросов;
- выбор вариантов ответа;
- автоматический подсчёт результатов;
- формирование списка подходящих профессий.

---

### 👩‍🌾 Каталог профессий
- список профессий агропромышленного комплекса;
- подробное описание каждой профессии;
- информация об уровне заработной платы;
- ссылки на внешние источники и видео.

---

### 🎓 Учебные заведения
- список образовательных организаций Ростовской области;
- отображение на интерактивной карте;
- ссылки на официальные сайты;
- связь с профессиями и специальностями.

---

### 🔧 Административная панель
- управление профессиями;
- редактирование тестов и вопросов;
- добавление статей;
- управление учебными заведениями и специальностями.

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/mashagulko/agroput.git
cd agroput
````

---

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

Активация:

**Windows:**

```bash
venv\Scripts\activate
```

**Linux / macOS:**

```bash
source venv/bin/activate
```

---

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

---

### 4. Применение миграций

```bash
python manage.py migrate
```

---

### 5. Создание суперпользователя

```bash
python manage.py createsuperuser
```

---

### 6. Запуск проекта

```bash
python manage.py runserver
```
