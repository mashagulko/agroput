# Путь в Агро

Профориентационный веб-навигатор для школьников, студентов и абитуриентов, предназначенный для выбора профессий агропромышленного комплекса и построения образовательной траектории.

---

## Описание проекта

«Путь в Агро» — это веб-приложение, разработанное на языке Python с использованием фреймворка Django. Система помогает пользователям:

- ознакомиться с профессиями агропромышленного комплекса;
- пройти профориентационный тест;
- получить рекомендации по выбору профессии;
- узнать, где можно получить соответствующее образование;
- изучить статьи и материалы по аграрной тематике.

Проект ориентирован на школьников 8–11 классов, студентов и абитуриентов, интересующихся сельским хозяйством, агрономией, ветеринарией, экологией и современными агротехнологиями.

---

## Основные функции

### Главная страница
- описание проекта;
- визуальный маршрут пользователя;
- статистическая информация;
- статьи об агропромышленном комплексе.

### Профориентационный тест
- последовательное отображение вопросов;
- выбор вариантов ответа;
- автоматический подсчет баллов;
- вывод наиболее подходящих профессий.

### Каталог профессий
- описание профессий;
- примерный уровень заработной платы;
- ссылки на дополнительные материалы.

### Учебные заведения
- список образовательных организаций Ростовской области;
- адреса и ссылки на официальные сайты;
- связь с профессиями и специальностями.

### Административная панель
- добавление и редактирование профессий;
- настройка тестов, вопросов и вариантов ответов;
- управление статьями и учебными заведениями.
